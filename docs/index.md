## Project Overview

Beer Pong League manages beer-pong leagues, tournaments, matches, and ELO rankings. It uses React, TypeScript, Vite, Tailwind, and Supabase, with optional offline support via localStorage.

## Quick Reference

- **Tech Stack:** React 18, TypeScript 5, Vite 5, Tailwind, Supabase, React Router 6
- **Entry Point:** `src/main.tsx` → `App.tsx`
- **Architecture Pattern:** Component-based SPA with BaaS; React Context for state
- **Database:** Supabase Postgres (see [data-models.md](./data-models.md))
- **Deployment:** Vercel (see [deployment-guide.md](./deployment-guide.md))

## Getting Started

### Prerequisites

- Node.js 18+
- npm
- Supabase project (URL + publishable key)

### Setup

```bash
npm install
```

Create `.env.local`:

```env
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_PUBLIC_KEY=sb_publishable_xxx
```

### Run Locally

```bash
npm run dev
```

### Run Tests

```bash
npm run lint
```

(No dedicated test runner in `package.json`; add as needed.)
